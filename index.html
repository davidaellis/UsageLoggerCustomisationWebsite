This website allows researchers to customise Usage Logger. It is possible to specify the type or types of data to be collected. The order of collection can also be configured. Click a section below to reveal more information

Select Data Sources

  • Contextual data provides a researcher with a snapshot of what software is installed on a smartphone. This can include apps installed, permissions requested and/or permissions granted.
  • Continuous logging records smartphone interactions after installation. Features captured include screen on/off events, specific app use and notifications received.
  • Past usage extracts data from an internal database of historical data maintained by Android.

Customise Order

Customise the order data is collected by dragging and dropping the text on the right hand of the screen that appears after data multiple data sources are selected.

Generate a QR Code

Clicking the 'finished' button will generate a QR code. This is required to programme the app. Participants can then scan the QR code from within Usage Logger, which will start data collection. The QR code can also be saved for future use.
